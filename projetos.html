Case Principal

FrotaCerta SaaS

Plataforma de inteligência financeira automotiva.

Acessar

SaaS desenvolvido do zero para resolver dores reais de proprietários de veículos. Futuramente o sistema irá oferecer controle total sobre custos de manutenção, abastecimento e prevê gastos futuros através de análise de dados.

Backend (API Restful)

  • Desenvolvido em Java 21 e Spring Boot 3.
  • Autenticação via JWT (Spring Security).
  • Integração com Stripe API para gestão de assinaturas e webhooks.
  • Banco de dados PostgreSQL.

Frontend (SPA)

  • Construído com Angular 21 (Standalone Components).
  • Visualização de dados complexos com Chart.js.
  • Design responsivo e consumo de API tipado.
Java Spring Boot PostgreSQL Docker Angular TypeScript Stripe Chart.js
